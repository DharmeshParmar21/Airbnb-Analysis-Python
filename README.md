## Project Overview
Airbnb is a global marketplace connecting travelers with hosts offering short-term accommodations.  
This project analyzes Airbnb listing data to understand pricing trends, room-type patterns, neighborhood insights, and review activity using structured EDA and visualizations.

## Files Included
- Dataset (CSV)  
- Jupyter Notebook (full code)  
- PDF Report  
- PPTX Presentation  
- Project Banner Image  

## Key Insights
- Prices vary widely with no strong concentration in any range.  
- Entire home/apt and private rooms dominate the listings.  
- Manhattan & Brooklyn have the highest listing volume.  
- Room-type pricing shows clear differences, especially for shared vs entire homes.  
- Review frequency changes over time with noticeable peaks and drops.  

## Business Recommendations & Suggestions
- Optimize pricing based on room type and neighborhood trends.  
- Improve visibility for listings in underrepresented areas.  
- Encourage hosts to use insights from review patterns for better engagement.  
- Apply dynamic pricing to stay competitive in high-demand zones.  

## Tools & Technologies Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook.

## How to Use
1. Load the CSV dataset.  
2. Run the Jupyter Notebook for cleaning, EDA, and visualizations.  
3. Refer to the PDF for results and explanations.  
4. View the PPTX for a summarized presentation.
